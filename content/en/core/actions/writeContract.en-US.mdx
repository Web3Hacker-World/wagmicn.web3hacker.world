# writeContract

Action for calling a Contract write method.

This is a wrapper around viem's [`writeContract`](https://viem.sh/docs/contract/writeContract.html).

```ts
import { writeContract } from '@wagmi/core'
```

## Usage

The following examples use the [wagmigotchi](https://etherscan.io/address/0xecb504d39723b0be0e3a9aa33d646642d1051ee1) contract.

```ts {4-8}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',
  abi: wagmigotchiABI,
  functionName: 'claim',
  args: [69],
})
```

### Prepared Usage

```ts
import { prepareWriteContract, writeContract } from '@wagmi/core'

const config = await prepareWriteContract({
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',
  abi: wagmigotchiABI,
  functionName: 'claim',
  args: [69],
})

...

const { hash } = await writeContract(config)
```

## Return Value

```ts
{
  hash: `0x${string}`,
  wait: (confirmations?: number) => Promise<TransactionReceipt>,
}
```

## Configuration

### abi

Contract ABI.

By defining inline or adding a [const assertion](https://www.typescriptlang.org/docs/handbook/release-notes/typescript-3-4.html#const-assertions) to `abi`, TypeScript will infer the correct types for `functionName` and `args`. See the wagmi [TypeScript docs](/core/typescript) for more information.

```ts {4}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  abi: wagmigotchiABI,
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',
  functionName: 'feed',
})
```

### address

Contract address.

```ts {5}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  abi: wagmigotchiABI,
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',
  functionName: 'feed',
})
```

### functionName

Name of function to call.

```ts {6}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',

```

### args (optional)

Arguments to pass to function call.

```ts {7}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',

  functionName: 'feed',
  args: [],
})
```

### chainId (optional)

Checks the current chain to make sure it is the same as `chainId`. If `chainId` is not the current chain, the connector attempts to switch to it before sending the transaction.

```ts {7}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',
  abi: wagmigotchiABI,
  functionName: 'feed',
  chainId: 1,
})
```

### account (optional)

The Account to write to the contract from.

```ts {7}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',
  abi: wagmigotchiABI,
  functionName: 'feed',
  account: '0xf39fd6e51aad88f6f4ce6ab8827279cfffb92266',
})
```

### gas (optional)

Gas limit for transaction execution.

```ts {7}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',
  abi: wagmigotchiABI,
  functionName: 'feed',
  gas: 1_000_000n,
})
```

### gasPrice (optional)

The price (in wei) to pay per gas. Only applies to [Legacy Transactions](https://viem.sh/docs/glossary/terms#legacy-transaction).

```ts {7}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',
  abi: wagmigotchiABI,
  functionName: 'feed',
  gasPrice: parseGwei('20'),
})
```

### maxFeePerGas (optional)

Total fee per gas (in wei), inclusive of `maxPriorityFeePerGas`. Only applies to [EIP-1559 Transactions](https://viem.sh/docs/glossary/terms#eip-1559-transaction)

```ts {7}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',
  abi: wagmigotchiABI,
  functionName: 'feed',
  maxFeePerGas: parseGwei('20'),
})
```

### maxPriorityFeePerGas (optional)

Max priority fee per gas (in wei). Only applies to [EIP-1559 Transactions](https://viem.sh/docs/glossary/terms#eip-1559-transaction)

```ts {7}
import { writeContract } from '@wagmi/core'

const { hash } = await writeContract({
  address: '0xecb504d39723b0be0e3a9aa33d646642d1051ee1',
  abi: wagmigotchiABI,
  functionName: 'feed',
  maxPriorityFeePerGas: parseGwei('2'),
})
```

Unique number identifying this transaction.

