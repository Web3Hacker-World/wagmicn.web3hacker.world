# signMessage

Action for signing messages with connected account.

This is a wrapper around viem's [`signMessage`](https://viem.sh/docs/actions/wallet/signMessage.html).

```ts
import { signMessage } from '@wagmi/core'
```

## Usage

The following examples use the message: `gm wagmi frens`.

```ts
import { signMessage } from '@wagmi/core'

const signature = await signMessage({
  message: 'gm wagmi frens',
})
```

## Return Value

```ts
string
```

## Configuration

### message

Message to sign with wallet.
