# watchNetwork

Action for subscribing to [network](/core/actions/getNetwork) changes.

```ts
import { watchNetwork } from '@wagmi/core'
```

## Usage

By default, the callback will be invoked when network changes.

```ts
import { watchNetwork } from '@wagmi/core'

const unwatch = watchNetwork((network) => console.log(network))
```

## Return Value

`unwatch` is a function that can be called to unsubscribe from the network change event.
