# getNetwork

Action for accessing network data, such as current connected chain and [connector chains](https://wagmi.sh/core/connectors/injected#chains-optional).

```ts
import { getNetwork } from '@wagmi/core'
```

## Usage

```ts
import { getNetwork } from '@wagmi/core'

const { chain, chains } = getNetwork()
```

<Callout type="info">
  Tip: If you want the functionality to switch networks, [check out
  `switchNetwork`](https://wagmi.sh/core/actions/switchNetwork).
</Callout>

## Return Value

```ts
{
  chain?: Chain & { unsupported?: boolean }
  chains: Chain[]
}
```
